# Todo App with API

## Demo

- [DEMO LINK](https://oleh-yablunovskyi.github.io/todo-app/)

## Description

This App represents a minimalistic list of your Todos.

The user can:
  * add new Todos
  * delete the old ones
  * edit the existing Todos
  * toggle status of each separate Todo or all at once

## Technologies used:

- React
- Javascript
- Typescript
- REST API
